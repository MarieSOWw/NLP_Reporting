📖 Documentation

Pipeline et technologies du projet

🎯 Objectif

Construire un système capable de générer des rapports business automatiques en langage naturel. Pipeline complet : Data → Analytics → NLP → Dashboard

Pipeline de traitement
1

Apache Spark

Chargement CSV, nettoyage, 12 KPIs calculés (CA, variations, régions, segments...)

2

NLTK

Tokenisation, NER, analyse de sentiment VADER, extraction de mots-clés

3

Mistral API

Génération de rapport NLP en langage naturel (FR/EN) via Mistral API

4

PostgreSQL

11 tables = source unique de vérité du système

5

FastAPI

20+ endpoints REST + chatbot RAG entre PostgreSQL et les consommateurs

6

Restitution

Dashboard web (10 graphiques), PDF (13 graphiques), Grafana, Audio, Chatbot RAG
